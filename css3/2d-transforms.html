CSS3 2D Transforms

Overview

Using translate to rotate, scale, and skew elements.

Demo

Code

  • html
  • CSS

<img src="path/to/your/image.png" />

img {
-ms-transform: translate(10px,10px) rotate(20deg) scale(0.5,0.5) skew(20deg, 10deg); /* IE 9 */
-webkit-transform: translate(10px,10px) rotate(20deg) scale(0.5,0.5) skew(20deg, 10deg); /* Safari */
transform: translate(10px,10px) rotate(20deg) scale(0.5,0.5) skew(20deg, 10deg);
}
